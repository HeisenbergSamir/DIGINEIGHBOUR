Built by Samir | @designdazzle

## Quick Deploy on Railway

1. Go to railway.app → New Project → Deploy from GitHub
2. Connect this repo
3. Add environment variables:
   - BOT_TOKEN = your telegram bot token
   - SECRET_KEY = any random string
4. Done. Bot starts automatically.

## Dashboard
Run `python dashboard.py` — opens on port 5556.
First visit: create your superadmin account.
Then: Add Community → Add Bot Token → Add Categories → Generate invite → Test!

## Commands for residents
/start — main menu
/help — help
/search — search listings  
/invite — generate invite link
/author — about DigiNeighbour
/contact — contact developer
/sponsor — advertise here
/cancel — cancel action
